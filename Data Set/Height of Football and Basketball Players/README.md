# Дасгал ажил болон шалгалтад зориулсан өгөгдөл

Хөл бөмбөг болон сагсан бөмбөгийн мэргэжлийн тамирчдаас энгийн санамсаргүй түүврийн аргаар цуглуулсан тамирчдын биеийн өндрийн (feet) мэдээлэл бүхий өгөгдөл.
